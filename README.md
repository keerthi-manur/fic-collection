A beautiful, locally-hosted catalogue for organizing and tracking your favorite fanfiction.

## Features

- **Browse Your Collection** - View all your favorite fics in an elegant library-inspired interface
- **Add Fics** - Store fic titles, links, fandoms, ships, tags, and personal notes
- **Track Rereads** - Mark every time you reread a fic to keep count
- **Edit Anytime** - Update fic information whenever needed
- **Filter & Search** - Filter by fandom or ship to find exactly what you're looking for
- **Dark Mode** - Elegant dark theme with warm gold and green accents
- **Local Storage** - All data is saved locally in your browser (no account needed)

## How to Use

### Option 1 - Github Page
1. Click on __________
2. That's it!

### Option 2 - Opening the Catalogue
1. Download the `index.html` file
2. Open it in your web browser (works in Chrome, Firefox, Safari, Edge, etc.)
3. That's it! No installation or setup needed

### Adding Fics
1. Click the **+ Add Fic** button
2. Fill in:
   - **Title of Work** - the fic name
   - **Archive Link** - the site URL
   - **Universe** - the fandom
   - **Ship** (optional) - the pairing(s)
   - **Themes & Elements** - comma-separated tags
   - **Personal Notes** - why you love it
3. Click **Add to Collection**

### Managing Your Collection
- **Reread** - Click the Reread button each time you reread a fic
- **Edit** - Click Edit to change any fic information
- **Delete** - Click Edit Collection to reveal delete buttons, then Remove to delete
- **Filter** - Use the dropdown menus to filter by Universe or Ship

### Importing/Exporting Data
Your data is stored locally in your browser's localStorage. If you want to use this catalogue in a different browser or device:
1. Export your collection as a backup
2. Import it in another browser
3. All your fics will appear

## Data & Privacy

- **All data is stored locally** - Nothing is sent to servers or shared online
- **Persistent storage** - Your collection stays saved even after closing the browser
- **Browser-specific** - Each browser has its own separate collection (use export/import to sync)
- **No tracking** - No analytics or user tracking of any kind

- Single-file application (no dependencies)
- Uses browser localStorage API
- Responsive design works on desktop and mobile

## Troubleshooting

**I added a fic but it disappeared**
- Check if you're using a different browser or private/incognito window
- Each browser stores data separately
- Use export/import to move data between browsers

**The page looks weird**
- Try refreshing the page
- Clear your browser cache
- Make sure you're using a modern browser

**I accidentally deleted something**
- Unfortunately there's no undo, but you can use a previous export backup

## Customization

You can customize the appearance by editing the CSS in the HTML file:
- Change colors in the `:root` or specific color values
- Modify fonts by changing the font imports
- Adjust layout by changing grid or flexbox properties

## Tips & Tricks

- Use specific, searchable tags to make filtering easier
- Add detailed notes about what makes a fic special to you
- Regularly export your collection as a backup
- Use the ship filter to find all your favorite pairings at a glance
